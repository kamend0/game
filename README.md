contributors of that project, has made my life so much nicer.


## Dev Environment / Workflow

I put together a few helpful shell commands which are stored in `commands/`. To 
run these in a sane manner, I highly recommend installed direnv. On Mac: `brew 
install direnv`. Then run `direnv allow` at the project root to enable the 
`.envrc` file to add the commands subdirectory to the path while in the 
project.

You need direnv installed to use the shell commands in `commands/`. `brew 
install direnv`. Update the `.envrc` file as necessary and run `direnv allow` 
to allow the new commands. Shorter, the better.
